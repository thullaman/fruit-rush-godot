# Fruit Rush

Fruit Rush is a 2D parkour-based platformer built using the Godot Engine. The project focuses on movement mechanics, level design, collectibles, and adding basic enemy interaction to create a more complete gameplay loop.

## Gameplay Overview

The game starts on a main menu where the player can choose between Level 1 and Level 2.

Each level is built around parkour-style platforming with jumps, obstacles, and collectible melons. Collected melons are counted and displayed on the top-left of the screen as a score.

Hidden easter-egg shortcut paths are placed in both levels, allowing players to discover easier ways to complete them if they explore carefully.

At the end of Level 1, touching the trophy transitions the player to Level 2. Completing Level 2 returns the player back to the main menu.

## Core Features

- Parkour-based 2D platforming movement
- Two fully playable levels
- Melon collection system with score counter
- Hidden shortcut (easter-egg) paths
- Enemy system (mushroom-based enemies)
- Enemy damage and bounce-back interaction
- Player jump boost when defeating enemies
- Fall detection logic
- Health system
- Pause screen with Resume and Main Menu options
- Clean level transition system

## Enemy & Health System

Enemies damage the player upon collision. When defeated correctly, the player bounces upward, encouraging skill-based interaction. A health system tracks player survivability, adding challenge to the gameplay.
